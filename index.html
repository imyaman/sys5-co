상품

SYSTEM5는 모바일, 클라우드, 플랫폼 서비스를 제공합니다.

모바일 앱 테스트

모바일 앱 테스트 서비스 : 모바일 앱이 다양한 디바이스에서 이상 없이 동작하는지 확인합니다.

모바일 앱 크래시 리포트

모바일 앱 크래시 리포트 서비스 : 모바일 앱이 사용자의 디바이스에서 비정상 종료될 경우 그에 대한 데이터를 수집합니다.

설치가능한 웹 애플리케이션 카탈로그

설치가능한 웹 애플리케이션 카탈로그 서비스 : 사용자가 쉽게 웹 애플리케이션을 찾을 수 있도록 웹 애플리케이션 카탈로그를 제공합니다.

mBaaS(mobile Backend as a Service)

모바일 백엔드 서비스 : 모바일 앱에 필요한 서버의 기능을 미리 만들어 API로 제공합니다.

모바일 앱 테스트 서비스를 이용해보세요. ucloud biz의 ucloud appster를 이용해보세요.
